# Online Job Portal

An online job portal web application built using React, Tailwind CSS, and Firebase Firestore. It enables users to search for and apply for jobs based on specific filters such as job title, location, job type, and experience. The platform is designed with a clean and responsive user interface, ensuring a seamless experience on both desktop and mobile devices.

## Features
- **Dynamic Job Search**: Filter jobs by title, type, location, and experience level.
- **Apply for Jobs**: Direct links to apply for job listings.
- **Real-time Data**: Display job posting information using Firebase Firestore.
- **Responsive Design**: Works well on all devices with the help of Tailwind CSS.
- **Modern UI**: Clean, user-friendly interface with smooth interactions.

## Technologies Used
- **React**: Frontend library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Firebase Firestore**: Real-time NoSQL database for job listings and user data.
- **Day.js**: A lightweight library for date and time manipulation.
- **Vite**: A build tool for faster development with React.
